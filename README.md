# 青铜密符 - 中国神话解密游戏

## 游戏简介

《青铜密符》是一款融合中国古代神话与刑侦推理元素的网页解密游戏。玩家将通过搜索关键词收集线索，解开谜题，揭开故事情节，最终阻止一场可能毁灭世界的灾难。

## 游戏特色

- **引人入胜的故事情节**：融合中国古代神话与现代刑侦推理元素
- **丰富的谜题系统**：包含凯撒密码、摩斯密码、逻辑推理等多种谜题类型
- **关键词搜索机制**：通过搜索关键词获取线索，推进剧情发展
- **线索管理系统**：收集、整理和分析线索，解开复杂谜题
- **精美的视觉设计**：采用深蓝、暗红和金色配色，营造神秘悬疑氛围

## 如何开始

1. 打开 `index.html` 文件即可开始游戏
2. 阅读故事情节，寻找金色下划线的关键词
3. 点击关键词或在搜索框中输入关键词获取线索
4. 解开谜题，推进剧情发展
5. 最终揭开故事的真相

## 游戏操作

- **搜索关键词**：点击故事中的金色关键词或在搜索框中输入关键词
- **查看线索**：在右侧线索墙查看已收集的线索
- **解开谜题**：当遇到谜题时，在弹窗中输入答案并提交
- **获取提示**：点击右上角的"提示"按钮获取游戏提示
- **返回上一节点**：点击顶部的"返回"按钮回到之前的故事节点

## 技术实现

- HTML5 + CSS3 + JavaScript (ES6+)

- Font Awesome：提供丰富的图标资源
- 纯前端实现，无需后端支持
